# Medical Insurance Claim Analysis

## Overview
This project focuses on analyzing medical insurance claim data to identify factors that influence insurance charges. The analysis helps uncover patterns and trends using data visualization and exploratory data analysis techniques.

## Features
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Interactive Data Visualizations
- Analysis of factors affecting insurance charges
- Streamlit-based web application

## Technologies Used
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Streamlit
